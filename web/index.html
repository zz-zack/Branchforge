BranchForge

Run AI engineering teams in parallel branches.

A Git-native Agentic Harness - Claude agents work like a whole engineering team, in parallel, across isolated git worktrees; integrate locally; ship one clean result. To the outside it looks like one developer; inside, it is a team.

Two ways to use it

forge run - variants

Run one task as N governed, test-verified parallel variants. Pick the best.

task
-> N isolated worktrees (parallel)
-> each: code -> test -> fix loop
-> budget + kill-switch
-> compare, you merge one

forge team - contract

Decompose a goal into contract-bound parts, build in parallel, integrate and self-heal.

goal + contract test
-> Lead splits into parts
-> parts built in parallel
-> merge, contract test gate
-> self-heal, one clean result

Quickstart

1
Make sure claude is logged in (or set ANTHROPIC_API_KEY), then install the one runtime dep:
npm i @anthropic-ai/claude-agent-sdk
2
Variants - run a task as 2 test-verified parallel variants under a budget:
node forge.mjs <repo> "<task>" --variants 2 --budget 0.5
3
Team - decompose a goal into contract-bound parts, integrate and self-heal (repo needs a contract.test.js):
node forge-team.mjs <repo> "<goal>" --budget 0.8
